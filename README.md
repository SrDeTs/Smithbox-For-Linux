# Smithbox for Linux

This is a Linux build of Smithbox, a modding tool for Elden Ring and other FromSoftware games.

## Running

```bash
cd  /linux-x64
./smithbox
```

Or simply run `./Smithbox` directly.

## Issues

For any issues or errors, please report to the official repository:

https://github.com/vawser/Smithbox?tab=contributing-ov-file

This is a Linux-only build.
